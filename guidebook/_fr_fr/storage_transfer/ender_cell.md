# Cellules de l'End

![](./ender_cell.png)

La cellule de l'End est un bloc utilisé pour stocker de l'énergie (FE) vers un canal spécifique du réseau de l'End à son propriétaire. 

Vous pouvez accéder à l'énergie stockée d'un canal sélectionné depuis n'importe où dans le monde si la cellule de l'End que vous voulez utiliser a un canal actif avec une capacité valide.
