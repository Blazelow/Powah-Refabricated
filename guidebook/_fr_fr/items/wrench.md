# Clé à molette

La clé à molette à 3 modes : 

Mode Configuration : utilisé pour changer les entrées/sorties des cables. 

Mode Liaison : utilisé pour lier les blocs entre eux comme les orbes et les barres. 

Mode Rotation : utilisé pour tourner les blocs horizontallement.
