# Réacteur

![](./reactor.png)

Le réacteur est un générateur multi-blocs (FE) qui utilise l'Uraninite comme combustible principal. 

Pour le construire, vous aurez besoin de 36 blocs de réacteur dans votre main et de les placer dans une zone 3X4 remplaçable, puis le réacteur se construira automatiquement.
