# Thermogénérateur

![](./thermo_generator.png)

Le thermogénérateur est un générateur à FE qui génère de l'énergie lorsqu'il est placé sur un bloc/fluide à haute température comme la lave. Nécessite un liquide de refroidissement comme l'eau pour fonctionner.
