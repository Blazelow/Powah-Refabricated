# Эндер-ячейки

![](./ender_cell.png)

Эндер-ячейка предоставляет стационарный доступ к уникальной для каждого игрока эндер-сети с раздельными каналами для хранения FE-энергии. 

Вы можете управлять энергией внутри вашей эндер-сети из любой точки мира (при условии доступности энергобуфера в активном канале эндер-ячейки).
